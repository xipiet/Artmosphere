# Artmosphere - Orakel

- Web-App mit /ipad und /main <br/>

## Installation

- git clone <br/>
- cd Artmosphere <br/>
- Node & npm installieren (https://nodejs.org/en/download) <br/>
- npm init -y <br/>
- npm install <br/>
(sollte express dotenv ldap-authentication node-fetch automatisch installieren, ansonsten manuell)  <br/>
- node server.js <br/>

## Update

- ps aux | grep node <br/>
- kill "entsprechende PID" <br/>
- cd /Artmosphere <br/>
- git pull <br/>
- reboot <br/>
